# 📱 SillyTavern 高保真手机 UI 扩展 (Phone UI)

这是一款为 SillyTavern (酒馆) 设计的沉浸式“小手机”渲染扩展。它可以将 AI 生成的特定格式文本，自动拦截并渲染为一个真实、可交互的智能手机聊天界面，支持发表情包、语音、转账、发送图片和撤回消息等丰富交互。

## ✨ 核心特性

- **高保真视觉渲染**：完美的双边气泡布局、仿微信/LINE的沉浸式界面。
- **丰富的媒体支持**：
  - 💬 文字消息
  - 🖼️ 图片与视频展示 (带有固定高度的描述滚动区)
  - 🎵 语音消息 (带有动态声波和时长)
  - 💰 转账卡片 (点击可“领取”并改变状态)
  - 📍 定位卡片
  - 撤回消息提示机制
- **动态交互注入**：你可以在界面上直接点击“+”号发送表情包、语音或转账，操作结果会无缝通过系统提示词注入 AI 的上下文，让 AI “看”到你的操作。
- **独立主题与表情包系统**：
  - 支持 **全局配置** 和 **角色专属配置**。
  - 主题和表情库支持打包并一键写入角色卡（PNG EXIF Extension 数据），发给别人也能立刻生效！
- **跨层聚合 (Aggregate)**：自动打通上下文，将上下多轮对话中的手机消息无缝拼接成一个连续的聊天记录界面。

---

## 📦 安装方法

1. 打开你的酒馆根目录，进入 `public/scripts/extensions/third-party/`。
2. 将本扩展文件夹（命名为 `shoujishouji-`）放入其中。
3. 刷新酒馆页面，点击顶部导航栏的“扩展插件(拼图图标)”，找到 **📱 Phone UI (小手机)** 菜单即可开启设置。

*(注：请确保文件夹名称与 `index.js` 中配置的 `FOLDER_NAME` 完全一致)*

## 📝 给 AI 设定的系统提示词 (Prompt)

要让 AI 输出手机界面，你需要将以下规则加入到角色的 System Prompt (系统设定) 或 Jailbreak (破狱) 中：

```markdown
当你和 {{user}} 使用手机聊天时，必须且只能使用以下严格的代码块格式。
手机记录必须包裹在 [phone:联系人备注] 和 [/phone] 之间，且必须另起一行，不要与其他正文混排。

[phone:备注名字]
# 支持的消息类型，一行一条，必须严格遵循前缀格式：
> 我: 普通文字消息
> 其他人: 对方回复的文字消息
> 其他人: [image] http://xxx.com/img.jpg|这里是图片的描述信息
> 其他人: [video] http://xxx.com/vid.mp4|这里是视频的描述信息
> 我: [voice] 12
> 银行: [transfer] 520
> 系统: [location] 所在位置的文字描述
> 其他人: [recall] (对方撤回了一条消息)
[/phone]
```

**书写规则说明**：
- 第一级的名字（如 `我:` 或 `其他人:`）必须和你设定在 `settings.html` 中的名字或酒馆当前角色名一致。目前系统主要通过识别 `我:`、`User:`、`{{user}}:` 来判断是否是右侧的绿色气泡。
- **图片/视频格式**：`[image] 链接|描述文字`。中间用竖线 `|` 分割。如果不填链接，只填描述，系统会自动为你渲染一个“没有加载出图片的底片效果”。
- **转账金额**：纯数字即可。

## 🎨 角色专属配置与导出

在扩展设置面板中，你可以管理你的手机主题和表情包。

### 1. 角色专属 vs 全局配置
扩展支持设置**全局默认**的表情包和主题皮肤。
当你加载一个特定角色时，你可以为该角色配置专属的皮肤（比如给这个角色换成粉色 LINE 主题）和专属的表情包映射。

### 2. 表情包格式 (JSON)
在文本框中输入 JSON 格式：
```json
{
  "摸头": "https://img.example.com/pat.gif",
  "流泪": "https://img.example.com/cry.png"
}
```
当你在手机界面点击“+”并发送 `摸头` 时，UI 就会自动渲染出这张图片。

### 3. 主题系统
你可以随时更改所有的手机UI颜色！只需提供 CSS `:root` 变量。
参考 `themes/THEME_GUIDE.md`（可直接发给 AI 让 AI 帮你生成新皮肤），并将生成的 CSS 粘贴到设置面板中的**角色专属主题**框内。

### 4. 写入角色卡 (Export)
配置好完美的专属主题和专属表情包后，点击 **“写入角色卡”**。这些配置会被自动存入角色卡的扩展数据区。
之后无论是分享这张 PNG 还是在新设备上加载，你的绝美手机 UI 配置都会跟随角色卡完美再现！

## 🛠️ 交互面板操作指引

当手机渲染在屏幕上后，注意底部有一个绿色的 **“+” 号输入条**：
1. **点击加号** 会弹出二级菜单：
   - **📸 图片/视频**：输入文字描述（或带链接），以媒体卡片形式发送。
   - **😊 表情**：输入你在 JSON 中配置的键值（如“流泪”），发送表情包。
   - **💰 转账**：输入数字进行转账。
   - **🎤 语音**：输入秒数，发送语音条。
2. 发送后，UI 会立即展示一个半透明的“预览气泡”。
3. 在你点击酒馆默认的发送按钮（继续对话）时，这些操作记录会自动被打包注入到 `System Note` 中发给 AI，告诉 AI：“用户刚才在手机上给你发了 520 块钱，并发送了一个【流泪】表情包”。
